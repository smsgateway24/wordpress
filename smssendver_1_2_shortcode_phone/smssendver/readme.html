plugin For WORDPRESS SMS OTP for SmsGateWay24

 

Installation through file-manager:

1. Unpack archive smssendver.zip in wp-content/plugins/

2. In the section plugins find /SMS OTP for SmsGateWay24/ and Press /Activate/

3. In the section /Settigns/ find /SmsGateWay24 Plugin/ - it's admin panel for general settings plugin SmsGateWay24

4. Enter shorcode [smsgateway24/] or [smsgateway24country/] use

 

Installation through admin panel WORDPRESS:

1. Enter the section /plugins/

2. Press ðàçäåë /Add new/

3. Press button /Upload plugin/

4. Press button /Browse.../

5. Upload archive smssendver.zip

6. Press button /Activate/

7. In the section /Settigns/ find /SmsGateWay24 Plugin/ - it's admin panel for general settings plugin SmsGateWay24

8. Enter shorcode [smsgateway24/] or [smsgateway24country/] use
